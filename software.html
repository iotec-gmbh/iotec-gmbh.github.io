Software

Software

Um Ihnen eine individuelle Lösung zu bieten, entwickeln wir für Sie eine maßgeschneiderte Softwarelösung, die mit einer für Sie angepasste Hardware kombiniert werden kann. Hierbei kann auf langjährige Erfahrungen im Bereich der Software sowie Firmwareentwicklung zurückgegriffen werden. So wird für die entwickelte Elektroniklösung ein auf den Anwendungsfall zugeschnittenes Programm geschrieben. Hierzu gehört auch das Erstellen eines angepassten Betriebssystems (Linux / Windows) sowie die Möglichkeit, die Anwendung mit intelligenten Algorithmen zu erweitern. Neben der Auswertung von Sensordaten stellt die Steuerung der Anwendung und Visualisierung der Ergebnisse mittels App oder Web-Portal eine weitere Kompetenz von iotec dar. Zur Speicherung der Daten werden relationale sowie nicht relationale Datenbanken eingesetzt, welche mit Algorithmen zur Auswertung kombiniert werden können. Wir unterstützen Sie von der Entwicklung bis zum Deployment (Software-Verteilung) und bieten auch das Hosting und Wartung von individuellen Lösungen an.
