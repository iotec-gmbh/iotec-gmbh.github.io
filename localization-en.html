Indoor Positioning Systems

Positioning  •  Interiors  •  Navigation


Indoor Positioning Systems determine the position of objects or people in buildings. Since GPS does not function reliably indoors due to the lack of visual contact with the GPS satellites, it is necessary to use a system that uses other technologies. For this purpose, WLAN, Bluetooth, UWB (UltrawideBand) or RFID (Radio-frequency Identification) can be used. iotec is using ultra-wideband technology which is a short-range radio technology with an accuracy better than 30cm and therefore significantly more precise than Bluetooth or WLAN localization. Differences in height can also be determined very accurately. UWB is based on the Time of Flight (ToF) runtime procedure. The light propagation time between an object and several receivers (anchors) is measured. For the exact localization of an object at least 3 receivers are necessary (trilateration).


iotec offers you the possibility to positioning systems that work indoors as well as outdoors. For indoors, the high-resolution UWB technology is used, which can than be supplemented outdoors with a GPS to achieve an efficient solution. This localization technique can be used for various industries and applications. The object that is to be located will be equipped with a tag that communicates with the anchors. The visualization of navigation and localization can be implemented via a web portal or an app.
